# Coming Out Static Site

Upload these files to a GitHub repository and enable GitHub Pages.

- `index.html` — site content
- `style.css` — responsive animated styling
- `script.js` — parallax, reveal animations, confetti
- `assets/depth-map.png` — generated depth-style grayscale map
- `assets/hero-background.jpg` + `assets/hero-foreground.png` — layered parallax hero

Edit the letter text inside `index.html` before publishing.
